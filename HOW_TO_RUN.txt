========================================

OPTION 1: USING INTELLIJ IDEA (RECOMMENDED)
============================================

Step 1: Open the Project
-------------------------
1. Launch IntelliJ IDEA
2. Click "Open" or File > Open
3. Navigate to and select the "GUI4" folder
4. Click "Open"

Step 2: Wait for Maven Import
------------------------------
1. IntelliJ will show a notification about Maven projects
2. Click "Load" or "Import" if prompted

3. Click "Apply" and "OK"

----------------------------

1. Navigate to: src/main/java/org/example/RUDonutsApp.java
2. Right-click on the file
3. Select "Run 'RUDonutsApp.main()'"

TROUBLESHOOTING
===============

Problem: "Cannot resolve symbol 'javafx'"
Solution:
- File > Invalidate Caches > Invalidate and Restart
- Wait for Maven to re-import dependencies

Problem: "Module not found: javafx.controls"
Solution:
- Make sure module-info.java exists in src/main/java/
- Rebuild project: Build > Rebuild Project

Problem: Application won't start
Solution:
- Check Run Configuration
- Edit Configurations > Add VM options:
  --module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml
- Or let IntelliJ handle it automatically (should work with pom.xml)

Problem: "Error: JavaFX runtime components are missing"
Solution:
- This means Maven dependencies didn't load
- Try: File > Reload All Maven Projects (right-click pom.xml)

Problem: Tests won't run
Solution:
- Right-click on pom.xml
- Maven > Reload Project
- Wait for dependencies to download
